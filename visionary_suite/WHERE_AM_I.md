## Symbol Language

### Prefix (WHO is asking):
- `~` = **play** - Exploration/gentle perspective
- `-` = **reader** - Observer/consumer perspective
- `^` = **create** - Builder/maker perspective
- `+` = **work** - Production/efficiency perspective
- `%` = **synergy** - Integration/harmony perspective

### Suffix (HOW it responds):
- `~` = **guided/gentle** - Menu-driven, tutorial
- `!` = **direct** - Fast, minimal explanation
- `?` = **questioning** - Interactive, clarifying
- `+` = **building** - Layered, educational
- `-` = **minimal** - Pure data, no decoration

## Troubleshooting

### Symbols not working?
Some shells have issues with `-` and `^` in aliases. Use:
```bash
listmancer -ls~   # Instead of -ls~
listmancer ^ls+   # Instead of ^ls+
\~ls\~            # Escape the symbols
```

### Commands not found?
```bash
source ~/.bashrc   # Reload configuration
type listmancer    # Verify function loaded
```

### Python errors?
```bash
which python3      # Verify Python 3 installed
python3 --version  # Check version
```

### Reset learning?
```bash
rm -rf ~/.listmancer/  # Fresh start
```
